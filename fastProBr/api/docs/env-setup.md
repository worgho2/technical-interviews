# Env Setup

1. Clone

```shell
git clone https://github.com/worgho2/fpbr-api.git
```

2. Install Dependencies

```shell
# Install triggers npm prepare script, installing husky hooks
npm i
```

3. Create env file

```shell
# Fill in the variables
cp .env.example .env
```

4. Start dev server

```shell
npm run dev
```

5. Start tests in watch mode

```shell
npm run dev:test
```
